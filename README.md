# 🎮 Arcade del Código

> Práctica de arreglos en JavaScript con una interfaz visual simple hecha con Bootstrap.

Este proyecto reúne varios retos de JavaScript para practicar manipulación de arreglos, ciclos, búsqueda, transformación y acumulación de datos. La página principal funciona como una pantalla de bienvenida estilo arcade y los resultados se ven en la consola del navegador.

## 📁 Estructura del proyecto

```text
practica-arreglos-for/
├── index.html        ← página principal con interfaz visual
├── reto-1.js         ← El inventario del explorador
├── reto-2.js         ← La fila del cajero automático
├── reto-3.js         ← Pasar lista
├── reto-4.js         ← Poción de doble poder
├── reto-5.js         ← El filtro de aprobados
├── reto-6.js         ← El detective de usuarios
├── reto-7.js         ← La caja registradora
└── torneo-final.js   ← El Torneo de la Academia
```

## 🚀 Cómo usarlo

1. Abre `index.html` en tu navegador.
2. Presiona `F12` y entra a la pestaña **Console**.
3. Revisa los resultados de cada reto directamente en la consola.

## 🛠️ Tecnologías usadas

- HTML5.
- CSS3.
- JavaScript ES6+.
- Bootstrap 5 vía CDN.
- Consola del navegador para mostrar los resultados.

## 🎨 Interfaz

La pantalla principal usa Bootstrap junto con estilos personalizados para mostrar un bloque tipo arcade con un aviso visible y centrado. Bootstrap se carga desde CDN en `index.html`, lo que facilita mantener el proyecto liviano y sin dependencias locales [web:142].

## 📚 Retos incluidos

- Índices y `.length`.
- Métodos de arreglo: `push`, `unshift`, `pop`, `shift`, `splice`.
- Ciclo `for`.
- `map()`.
- `filter()`.
- `includes()` e `indexOf()`.
- `reduce()`.
- Uso combinado de arreglos paralelos, `sort()` y `filter()` en el reto final.
